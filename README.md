## Portfolio Website

This is a personal portfolio website for **Shivam Gaur**, a BCA student and **MERN Stack Developer**.  
It showcases skills, featured projects, and provides quick links to your profiles.

## [View Demo](https://dev-shivamgaur.github.io/portfolio/)  

## Features

- **Responsive hero section** with your name, title, and call-to-action buttons.
- **About section** describing your background and interests.
- **Skills grid** listing core technologies like HTML, CSS, JavaScript, Node.js, Express.js, MongoDB, and more.
- **Projects section** highlighting work such as the URL Shortener and Portfolio repo.

## Tech Stack

- **HTML5**
- **CSS3** (custom styling, responsive design)
- **JavaScript (ES6+)**

## Getting Started

- **Clone or download** this project.
- Open `index.html` directly in your browser  
  or serve it using any static server (e.g. VS Code Live Server).

## Customization

- Update text content in `index.html` (name, description, skills, projects).
- Add or modify project cards in the **Projects** section.
- Adjust styles in the `<style>` block inside `index.html` to tweak colors, fonts, and layout.
